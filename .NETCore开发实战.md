 课程介绍   .NET Core 已经被微软视作 .NET 未来的发展方向，.NET Core 与以往版本最大的不同就是跨平台和开源。跨平台意味着你可以有更多的开发环境和部署环境的选择，尤其是对 Docker 和 Kubernetes，.NET Core 都具有良好的支持，开发者可以基于.NET Core 快速构建微服务架构并部署到 Kubernetes 云基础设施中，并且实现高可用、可伸缩的系统架构搭建。同时，由于其开源的性质，开源社区也贡献了大量的 .NET Core 核心代码，各类主流组件库也都有对.NET Core 的支持，这样，开发者就可以更多的关注业务设计与实现，快速实现商业价值。

 .NET Core 云原生微服务架构的开发，不仅涉及到.NET Core 重要组件的知识，还涉及到 DDD、远程调用 RPC、熔断限流、网关、身份认证、安全等微服务架构的各个方面，同时也要求技术人员对 DevOps 协作模式有一定的掌握。
