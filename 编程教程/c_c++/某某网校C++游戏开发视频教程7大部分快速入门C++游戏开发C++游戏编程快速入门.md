 课程介绍：

 如果你会一点C、C++语言，你就可以学习游戏编程了，开发真正的游戏！如果你学过一点C++更好，没学过也没关系。

 本课程教你从零基础开始开发7个完整的游戏:Brainiac、Light Cycles、Henway、Battle Office、Meteor Defense、Space Out、Stunt Jumper。
