## 课程简介

<img src="https://ziyuanyun.oss-cn-guangzhou.aliyuncs.com/yun/20240515185155/664493cbcc233.jpg" width="500" alt="">

  React Native 是目前流行的跨平台移动应用开发框架之一。课程帮助同学们从React开始实战学习到React-Native的项目，做了一个大满贯的全程项目，也是目前使用率和开发频率非常高的项目，可以帮助缺乏项目经验的同学们极大的提升项目经验。React Native 的核心语言是 JavaScript，特别是 ECMAScript 6（ES6）。因此，需要熟悉ES6的一些最新功能。此外，在开始开发第一个 React Native 应用之前，需要安装必备的工具，比如 IDE，JavaScript 库等。另外，还需要了解核心的 React Native API。
