 课程大纲：

 SQL 介绍和 MySQL 设置我们将从快速介绍 SQL 环境和常见的“风格” （MySQL、SQLite、PostgreSQL 等）开始，然后下载并安装访问和探索 MySQL 数据库（MySQL 社区服务器）所需的工具+ MySQL 工作台)

 SQL 数据库基础知识 （第 1 部分）安装工具后，我们将快速回顾一些数据库基础知识并开始探索 MySQL 数据库中包含的16 个表，其中包含有关 Maven Movies 客户、库存和交易的信息。

 使用 MySQL 分析来自单个表的数据在本节中，我们将开始探索和从 MySQL 数据库中的各个表中提取信息，并练习编写 SQL 查询来选择、过滤、排序和分组我们的数据。[中期课程 MYSQL 项目]

 SQL 数据库基础知识（第 2 部分）课程的后半部分是关于在 MySQL 中分析来自多个表的数据，因此我们将从快速回顾关系数据库和数据库分析开始：主键与外键、关系基数、规范化等。

 通过 MySQL JOINS 分析多个表在本节中，我们将回顾最常见的 SQL 连接类型（INNER、LEFT、RIGHT、OUTER 等），并探索一些更复杂的 MySQL 查询来分析连接数据库中多个表的数据。[最终的 MYSQL 项目]

 重要提示： 本课程不包括 构建、更改或维护SQL/MySQL 数据库。我们的目标是专注于使用 MySQL 和 MySQL Workbench 进行数据探索和数据库分析（我们将在单独的课程中介绍数据库管理工具）。
