  【资源介绍】：

 一句话介绍：该课程分为四大块，分别是Git 的常用操作、IDEA 集成 Git GitHub Gitee GitLab、服务器上的 Git以及GitLab。

 Git 的常用操作：跟踪、提交、查看、修改、撤销、忽略、删除、回滚、前进、HEAD指针、引用日志、重置、标签、分支、冲突、推送、拉取、抓取等。

 IDEA 集成 Git GitHub Gitee GitLab：Git的常用操作在IDEA中的具体使用。

 服务器上的 Git：基于不同协议（文件协议、SSH协议、HTTP协议、GIT协议）下的Git服务器。

 GitLab：GitLab的安装、用户管理、群组管理、项目管理、邮件服务器整合、添加令牌、集成IDEA等。
