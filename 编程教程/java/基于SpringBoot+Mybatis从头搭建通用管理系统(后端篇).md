  基于SpringBoot+Mybatis从头搭建通用管理系统(后端篇)



 从零开始搭建完整的通用管理后端系统，使用Springboot+Mybatis搭建通用管理系统实例，使用mysql作为后端数据库，使用Spring Security作为权限控制框架，使用actable实现数据表自动创建，可以此项目为基础进行扩展开发，快速搭建管理系统，具体内容如下：

 用户登录（用户名密码登录、手机验证码登录、二维码扫码登录）。

 找回密码（邮箱找回、短信验证码找回）。

 系统主页。

 基础的增删改查功能及数据导出、权限管理、数据可发现校验、缓存管理、日志管理等。

 管理系统的常用功能：用户管理、系统配置、数据字典、日志管理、菜单管理、角色管理、消息管理、验证码管理、实体配置等基础功能。
